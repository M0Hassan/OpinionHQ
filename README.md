## OPinionHQ

#### Introduction

> OpinionHQ is a blog application where users can be able to post their opinions on any subject area.

#### Installation
* git clone https://github.com/M0Hassan/OpinionHQ
* cd OpinionHQ/OpinionHQ
* Run run.py 

#### Technology Stack Used 

##### Frontend
* HTML5 
* CSS3 
* BOOTSTRAP

##### Backend
* Python - Flask
* MySQL
